# README

## How to run sample program by CMake

1. `make clean`
2. `sh compile.sh`
3. `sh run.sh` 
4. Check `log.txt`

## How to run sample program by Makefile

This is based on main-V4 in [sample-Fortran-make](https://github.com/i16777215/sample-Fortran-make)

1. `make clean`
2. `make dir`
3. `make`
4. `sh run.sh` 
5. Check `log.txt`

## How to clean

1. `make clean`

## How to debug makefile

1. `make echo`
